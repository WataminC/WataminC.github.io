    Cancellation law

    Theorem

    If gcd(c, m) = 1 and

    ac≡bc(modm) ac \equiv bc \pmod{m} ac≡bc(modm)

    then

    a≡b(modm) a \equiv b \pmod{m} a≡b(modm)

    Where gcd shorts for the greaster common divisor.

    提示

    By definition of congruence, we have m∣(ac−bc)m | (ac-bc)m∣(ac−bc), equivalently, m∣(a−b)cm | (a-b)cm∣(a−b)c. Since gcd(c,m)=1gcd(c, m) = 1gcd(c,m)=1, it follows that m∣(a−b)m | (a-b)m∣(a−b), so as claimed.

    When a set SSS has an equivalence relation on it, then the equivalence relation partitions the set SSS into disjoint subsets, called equivalence classes, defined by the property that two elements are in the same equivalence class if they are equivalent.

    Congruence classes modulo m

    The set of congruence classes modulo m is denoted by Z/mZZ/mZZ/mZ. There are exactly m congruence classes in Z/mZZ/mZZ/mZ. That is:

    Z/mZ={[0]m,[1]m,⋯ ,[m−1]m} Z/mZ = \{[0]_m, [1]_m, \cdots, [m-1]_m\} Z/mZ={[0]m​,[1]m​,⋯,[m−1]m​}

    If [a1]m=[a2]m[a_1]_m = [a_2]_m[a1​]m​=[a2​]m​ and [b_1]_m = [b_2]_m, then

    [a1±b1]m=[a2±b2]m [a_1 \pm b_1]_m = [a_2 \pm b_2]_m [a1​±b1​]m​=[a2​±b2​]m​

    Notations of Congruence classes modulo m

    • Any element bbb of a congruence class [a]m[a]_m[a]m​ is called a representative of that class.
    • The set of all the least nonnegative representative of Z/mZZ/mZZ/mZ is the set of integers {0,1,2,⋯ ,m−1}\{0,1,2, \cdots,m−1\}{0,1,2,⋯,m−1}, that is called the least residue system modulo m.
    • Any set of m integers, no two of which are congruent modulo m, is called a complete residue system modulo m.
